## 10171 고양이

파이썬에서 __'\\\\'__ 는 역슬래쉬(\\) 를 출력할 때 사용

```python
print("\    /"+ "\\")
print(" )  ( ')")
print("(  /  )")
print(" \(__)|")
```

## 10172 개

파이썬에서  __\\"__ 는 큰따옴표(")를 출력할 때 사용

```python
print("|\_/|")
print("|q p|   /}")
print("( 0 )\"\"\""+"\\")
print("|\"^\"`    |")
print("||_/=\\\\__|")
```
